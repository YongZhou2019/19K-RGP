# 19K-RGP

The 19K rice genome project (19K-RGP) is a collaborative project to reveal rice genetic diversity to ensure food security for future generations. 

Here, we have posited our workflows, pipelines, scripts, codes, and all records for 10K-RGP, including:

1. HPC-GVCW workflow for variant calling, which refers to another page: https://github.com/IBEXCluster/HPC-GVCW and our previous publication (DOI: https://doi.org/10.1186/s12915-024-01820-5)
2. Venn plots, permutation plots;
3. Anything else that is associated with the Manuscript.
